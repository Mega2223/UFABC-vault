## Definição

A ROM é uma unidade de memória imutável, onde a informação colocada pelo fabricante é essencialmente permanente e só pode ser acessada. Uma ROM com $\large k$ sinais de entrada e $\large n$ sinais de saída é dita uma ROM $\large k \times n$. Os sinais de entrada são o endereço de uma determinada palavra e a saída é o conteúdo dessa palavra, logo, uma ROM comporta, no máximo $\large 2^k$ palavras (por meio da [[Codificação Digital|codificação]] de cada endereço), cada palavra com $\large n$ dígitos binários.
