## Funções 'simples'
Seja $c \in \mathbb{R}$ uma constante, $f$ e $g$ funções [[Limite#Continuidade|contínuas]] e [[Derivada|diferenciáveis]]:

| $h(x)$           | $h'(x)$                                         |
| ---------------- | ----------------------------------------------- |
| $cx$             | $c$                                             |
| $x^c$            | $cx^{c-1}$                                      |
| $e^x$            | $e^x$                                           |
| $(f + g)(x)$     | $(f' + g')(x)$                                  |
| $(f.g)(x)$       | $f'(x).g(x)+g'(x).f(x)$                         |
| $(f/g)(x)$       | $\large \frac{f'(x).g(x) - f(x).g'(x)}{g(x)^2}$ |
| $(f\circ g)'(x)$ | $f'(g(x)) .g'(x)$                               |
