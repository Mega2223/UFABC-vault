## Definição

Certas equações não são diretamente funções pois não cumprem com a propriedade de só haver um dependente para cada número de seu domínio, por exemplo a do círculo unitário:
$$\large x^2+y^2=1$$

Existem múltiplos valores $\large y$ para um determinado $\large x$. Para [[Derivada|diferenciar]] esta equação então se usa a derivação implícita.

## Definição

Para diferenciar equações, deve-se considerar a equação como um sistema de função de duas variáveis:

$$\large \frac{d}{dx}(x^2 + y^2) = \frac{d}{dx} 1$$
Pela propriedade distributiva
$$\large 2x + \frac{d}{dx} 2y = 0$$
Dividindo ambos os lados por $\large 2y$:

$$\large\begin{gather}
- \frac{2x}{2y}= \frac{d}{dx}
\end{gather}$$//ITITOODAPJFIP
